# 中國暗棋/Chinese smallXiangqi


## 中文

中國暗棋主要為 Jquery 開發，用 Node 啟動伺服器。

安裝伺服器相關 modules。

```
$ npm install
```

啟動伺服器

```
$ npm run dev
```

可以與玩家一對一，享受中國暗棋的樂趣。

如果你喜歡，請給我一顆星，感謝你。

## English

This website is Developed by Jquery.

Run Server by node.

We need to install the modules and run server.
