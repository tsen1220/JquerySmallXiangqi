## 中文

中國暗棋主要為 Jquery 開發，用 Node 啟動伺服器。

安裝伺服器相關 modules。

```
npm install
```

啟動伺服器

```
npm run dev
```

可以與玩家一對一，享受中國暗棋的樂趣。

## English

This website is Developed by Jquery.

Run Server by node.

We need to install the modules and run server.

```
npm install

npm run dev
```

Play with others. Enjoy the game!
